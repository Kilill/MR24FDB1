31 Starting code: 1 Byte = 0x55 not counted in length but counted in crc computation
32 Length: 2 Bytes low byte first = Lenght (2) + Function class (1) + Func group (1) Function (1) + Number of data bytes (n) + Crc (2) =
33  7 + n
34 Function Class: 1 byte (Function Code)
35 Function Group: 1 byte (Address 1) Function clasification
36 Function: 1 Byte (address 2) Specific function
37 Data: n Bytes of data
38 Crc: 2 Bytes low byte first
39 
40 crc is calculated over the complete packet including header but ofc exluding crc bytes
41 be aware that "Length" does include the crc bytes but does not count the header
